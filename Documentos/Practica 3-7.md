En este programa se imprimira el tipo de datos ya sean cadenas, enteros o booleanos
cadenas = ["Juan", "Pedro", "Chuy"]
```
print(cadenas)
print(cadenas[1])
print(type(cadenas))
print(type(cadenas[1]))
```
enteros = [5, 4, 8]
```
print(enteros)
print(enteros[0])
print(type(enteros))
print(enteros[0])
```
booleanos = [5, 4, 8]
```
print(booleanos)
print(booleanos[0])
print(type(booleanos))
print(booleanos[0])
```
